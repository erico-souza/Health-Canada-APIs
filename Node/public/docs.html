Documentation

Build Queries

All endpoints can be accessed at ### https://node.hres.ca. Health Canada APIs (v1.1.0) currently only support GET requests with support for the Lucene Query Syntax. The querystring can accept the following parameters search, count, skip & limit defined below:

Search:

Search for general keywords using search=KEYWORD. You can search for keywords in specific fields using the following syntax search=FIELD:KEYWORD. To search for multiple specific or general keywords join them using the AND or OR operators (search=FIELD:KEYWORD1+AND+KEYWORD2). You can access embedded fields using the . operator. You can limit a field by range using search=FIELD:[DATE1+TO+DATE2] or search=FIELD:[VALUE1+TO+VALUE2].

Count:

Aggregate on a field and return either term or histogram counts using count=FIELD. Term aggregration and histogram aggregation is determined automatically based on the fields value. For histogram aggregation fields, possible time intervals include year, quarter, month, week, or day. (count=HISTOGRAMFIELD:INTERVAL). The interval defaults to day. You can access embedded fields using the . operator. For term aggregation, add .keyword after the field. (count=FIELD.keyword).

Skip:

A number used to offset the the first result. Defaults to skip=0.

Limit:

A number used to limit the maximum number of documents returned. Defaults to limit=10. Max value is 1000.

Examples ###

Search for documents where the patient's gender is female.
?search=patient_gender:female Try it out

Search for documents where the patient's gender is female and a the outcome was recovered.
?search=patient_gender:female+AND+outcome:recovered Try it out

Search for documents where the report date is between 1997 and 2003.
?search=datintreceived:[1997+TO+2003] Try it out

Search for documents where the report date is between 1980 and 1990 and count the reports by year.
?search=datintreceived:[1980+TO+1990]&count=datintreceived:year Try it out

Count documents by the suspect report drug.
?count=report_drugname_suspect.keyword Try it out
